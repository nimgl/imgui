### Jp locale test program

#### Build

---

- Build "imgui_test.exe" --- Test GUI program

   ```sh
    $ make
   ```

- Build executables for test and "cimgui.dll"

   ```sh
    $ make test
   ```

- Build  "cimgui.dll" that statically linked with libstdc++-6.dll and etc

   ```sh
    $ make dll
   ```
